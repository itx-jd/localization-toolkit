CSV Generator

Create a starter CSV with a greeting key and auto-translate formulas for Google Sheets. Edit in your sheet, then export back to CSV.

Tip: The file contains =GOOGLETRANSLATE formulas for target columns.

How it works

The generated CSV contains columns: key, your base language, and chosen target languages. Target cells use GOOGLETRANSLATE($B2;"base";"target") so translations fill automatically when opened in Google Sheets.

After editing values, export as CSV and use the converter on the next tab to produce .arb files (e.g. app_en.arb, app_es.arb) inside a ZIP.

CSV → ARB Converter

CSV Format

Example header: key,en,es,ur. Each row maps a key to translations. Empty values are ignored. Output files are named app_<lang>.arb.

Tip: Keep keys snake_case (e.g. welcome_message). Avoid quotes in values. For placeholders, keep ICU style (e.g. {count}).
